Mounty Hall Problem Coding assignment

Requirements to run this project in local:
.Net 6.0
>=Node 14

Steps to run this project:

Execute the following command in "MountyHall/MountyHall.Tests" path run tests the backend
> dotnet test

Execute the following commands in "MountyHall/MountyHall.API" path to run the backend
> dotnet build
> dotnet run

Execute the following commands in "MountyHall/MountyHall.UI" path to run the frontend
> npm install
> npm start


Open browser and access url
UI : http://localhost:3000
